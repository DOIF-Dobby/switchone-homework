### 개발 환경

- Java 17
- Spring boot 3.3.0
- JPA

### API 문서 생성

```shell
./gradlew asciidoctor
```

### 어플리케이션 부트

```shell
./gradlew bootRun
```

### 접근 URL

- API 문서: `http://localhost:8080/docs/api-guide.html`
- h2 콘솔: `http://localhost:8080/h2-console`
